## 1.Result 对返回数据的一种封装
Result封装类中包含三个部分内容，如下：
1. **flag：** 返回状态，布尔类型，标志着返回成功或失败；
2. **code：** 返回码，int类型，自定义代码，标记返回内容是特殊需要的。
3. **message：** Object类型，返回信息。
